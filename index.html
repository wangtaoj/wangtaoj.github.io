mysql日期函数使用总结

发表于 2018-02-14

获取函数

mysql默认的时间格式: yyyy-MM-dd 或者 yyyy-MM-dd HH:mm:ss


1. Date()

返回日期部分, date(‘2018-02-14 17:03:04’) = ‘2018-02-14’

2. Time()

返回时间部分, time(‘2018-02-14 17:03:04’) = ‘17:03:04’

3. CurDate()

返回当前时间的日期部分

4. CurTime()

返回当前时间的时间部分

5. now()

代表当前时间, 包含日期+时间

6. Year(), Month(), Day(), Hour(), Minute(), Second()

返回一个指定日期的年, 月, 天, 小时, 分钟, 秒数

整型格式, 参数为一个指定的时间

7. DayOfWeek()
返回一个指定日期这一天是该周的第几天, 整型格式

注意: 星期日是第一天, 依次类推.

计算函数

1. DateDiff(‘’, ‘’)

参数: 两个时间

返回: 返回两个时间的日期的差值,整型数据。代表两个参数时间隔了多少天

如 datediff(‘2018-02-14 17:03:04’, ‘2018-02-13 18:23:12’) = 1

2.AddDate(‘’, interval expr type) || AddDate(‘’, days)

往指定的类型增加(加一天, 年, 月, 小时, 分钟, 秒等)

type: year, month, day, hour, minute, second等

例如:

adddate(‘2018-02-14 17:03:04’, 2) = ‘2018-02-16 17:03:04’

adddate(‘2018-02-14 17:03:04’, -2) = ‘2018-02-12 17:03:04’

adddate(‘2018-02-14 17:03:04’, interval -1 year) = ‘2017-02-14 17:03:04’

3.date_add(‘’, interval expr type)

用法与AddDate()类似.

Object clone()方法

打开Object类, 我们可以看到clone方法是一个本地方法。也就是说该方法是由java虚拟机实现的。
但是可以看到该方法的注释,该方法应该满足以下几点。
1) x.clone()!=x 原对象不等于克隆对象,相当于两个引用
2) x.clone().getClass() == x.getClass() 克隆对象与原对象的类型是一致的。
3) x.clone().equals(x) == true

遗传算法

发表于 2017-05-01

遗传算法简介

遗传算法（英语：genetic algorithm (GA) ）是计算数学中用于解决最佳化的搜索算法，是进化算法的一种。进化算法最初是借鉴了进化生物学中的一些现象而发展起来的，这些现象包括遗传、突变、自然选择以及杂交等。它的思想主要根据达尔文生物进化论而来,认为物种都是向好的方面去发展,所以当等到足够多的代数时,得到的值可与实际的值想接近。

算法步骤

1) 随机初始化一个种群
2) 个体评价,根据适应函数或者目标函数计算个体的适应值
3) 迭代设置
4) 个体选择,优胜劣汰。将好的基因的个体进入到下一代,作为父代种群。
5) 交叉算子,根据交叉概率pCross来判断父代个体是否需要进行交叉操作。一般取0.9,根据具体的问题设置。
6) 变异算子,根据变异概率pMutation来判断父代个体是否需要进行变异操作。一般取0.1, 主要作用是用来防止种群陷入局部最优解。
